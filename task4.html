Task 4: Grid Rows

What does auto do in grid rows?

auto adjusts the row height based on the content inside it.
Which row grows based on content?

The row with more content grows automatically.
Can rows have different heights?

Yes, grid rows can have different heights.
What happens if content is very large?

The row expands to fit the content.
Where are grid rows useful?

Grid rows are useful for headers, content sections, and footers.
