Task 1: CSS Grid Introduction

Box 1 – Difference between normal div and grid container

A normal div places items one below another.
A grid container allows items to be arranged in rows and columns.
Box 2 – Why does the layout look the same after applying Grid?

Because no rows or columns are defined yet.
Box 3 – Are grid items created automatically?

Yes, all direct child elements of the grid container become grid items automatically.
Box 4 – Can Grid work without rows or columns?

Yes, Grid can work without defining rows or columns, but no visual change will appear.
Box 5 – What does Grid need to show visible changes?

Grid needs rows, columns, or spacing like gap to show visible changes.
