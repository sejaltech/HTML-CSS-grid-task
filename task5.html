Task 5: Grid Gaps

Where does gap add space?

Gap adds space between grid items, not around the container.
How is gap different from margin?

Gap is controlled by the grid container, while margin is applied to individual items.
Does gap affect container size?

No, gap does not increase the outer size of the container.
Can we use gap only for rows or columns?

Yes, we can use row-gap or column-gap separately.
Why is gap preferred in Grid layouts?

Because it provides clean and consistent spacing between items.
