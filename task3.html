Task 3: Fixed vs Flexible Columns

Which layout breaks faster on small screens?

The fixed width (px) layout breaks faster on small screens.
Which layout adjusts automatically?

The flexible layout using fr adjusts automatically.
Why is fixed layout risky?

Because it does not adapt well to different screen sizes.
Where can fixed columns be useful?

Fixed columns are useful for sidebars or fixed size elements.
Which unit is best for responsive websites?

fr unit is best for responsive websites.
